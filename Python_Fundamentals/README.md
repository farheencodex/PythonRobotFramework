## üìù How to Use This Course

1. **Start from Topic 01** and progress sequentially ‚Äî each topic builds on prior knowledge.
2. **Read the comments** in each `.py` file ‚Äî they explain every concept in detail.
3. **Run each file** to see the output and understand behavior.
4. **Modify the practice tasks** ‚Äî experiment and break things to learn!
5. **Some files contain `input()` calls** ‚Äî these are commented out with hardcoded alternatives so files run without interaction. Uncomment them to try interactive mode.
